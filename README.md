# Next Dr.?

A daily medical terminology puzzle game. Guess the term from a set of
escalating clues — vague to obvious — organized by medical specialty.

## Tech stack
- Backend: Flask
- Database: PostgreSQL + SQLAlchemy
- Frontend: HTML/CSS
- Deployment: Render

## Status
In early development — currently building out core Flask routes.

## Running locally
1. Clone the repo
2. Create and activate a virtual environment
3. `pip install -r requirements.txt`
4. `python app.py`
